[게임서버 실행 가이드]
1. MMOServers.bat 배치 파일을 실행합니다.
2. 처음 실행할때는 각 서버 바이너리 관련 설정을 허용 해야합니다.
3. DummyClient.exe을 실행합니다. (DummyClient는 여러번 실행 가능합니다.)
4. 방향키, 채팅을 하면서 테스트 합니다.
(여러개의 클라이언트로 이동 시 게임서버 로그를 통해 AOI 확인 가능합니다.)
5. 길찾기 및 몬스터 관련 테스트는 게임서버 로그의 몬스터 좌표 체크 후
   커서로 좌표이동을 하면서 몬스터가 추적했을때 커서이동을 빠르게 하면 확인 가능합니다.
